idea连接mysql错误

尝试在idea上连接mysql，显示时区错误

解决：
1.先要保证系统环境变量里有mysql
2.cmd
3.mysql -hlocalhost -uroot -p
4.输入密码
5.show variables like’%time_zone’;
6.然后会显示时区为跟随系统
7.set global time_zone = ‘+8:00’;
8.转为东8区
9.然后再次尝试连接数据库
